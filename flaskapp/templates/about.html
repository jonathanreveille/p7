Ce projet, le fameux 7, fait partie du parcours DA Python d'Openclassrooms.
Le but étant de développer une application web avec Flask ayant pour but
de poser une question sur un lieu.

Grandpy se chargera de retourner un résumé de ce lieu (avec API de MediaWiki)
et un point géographique sur une carte par le biais de API Google Maps JavaScript.

L'application est codé en Python 3.7. Une grande partie du projet a été développé en
respectant la discipline du Test Driven Development.
